📂 How to Use Group Collections

  • Create Collection: Enter a collection name and click "Create Collection"
  • Add Groups: After creating, immediately select Facebook groups to add
  • Edit Existing: Click any collection on the right to edit its groups
  • Use Collections: Use collections in "Post to Groups" for targeted posting
  • Delete Collections: Click the delete button to remove unwanted collections

💡 Tip: Collections help you organize Facebook groups for different posting strategies.
